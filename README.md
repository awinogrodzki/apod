# Astronomy Picture Of The Day

## Setup

### Requirements

This application requires following binaries:
* node version 12
* npm version 6

### Installation

To install dependencies run `npm install` in project directory

## Development

### Running
In order to start development server run `npm start` in project directory

The server should be available at [http://localhost:8080](http://localhost:8080)

### Testing
In order to test the application run `npm test` in project directory
